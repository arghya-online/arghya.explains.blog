How to Make a Line Following Bot Using Arduino Uno

Published on: March 17, 2025

Introduction

In this blog, I will show you a simple way to build a line following bot using an Arduino Uno. This project is perfect for beginners who want to learn about robotics and basic electronics. I have explained every step in a simple manner so that anyone can follow along.

Components Required

  • Arduino Uno board
  • Robot Chassis (a basic frame for your robot)
  • 2 DC Motors
  • Motor Driver (such as L298N)
  • Infrared (IR) Sensor Module
  • Breadboard and Jumper Wires
  • Battery Pack

Step 1: Assembling the Chassis

First, assemble the robot chassis. Follow the instructions that come with your chassis kit. Mount the DC motors on the chassis and secure them properly. Make sure that the wheels are firmly attached and can rotate without any problems.

Image: Assembling the chassis

Step 2: Wiring the Motors and Motor Driver

Next, connect the DC motors to the motor driver. The motor driver controls the speed and direction of the motors. Use jumper wires to connect the motor driver to the Arduino. Make sure to follow the wiring diagram provided in the motor driver’s manual. This step is very important, so double-check your connections.

Image: Wiring the motor driver

Step 3: Integrating the IR Sensor Module

Now, attach the IR sensor module to the front of your robot. This sensor will detect the line on the ground. Connect the sensor’s output to one of the Arduino’s digital input pins. Position the sensor so that it faces the ground and can clearly see the line.

Image: Connecting the IR sensor

Step 4: Programming the Arduino

It is time to program the Arduino. Open the Arduino IDE on your computer and write a simple program that reads the sensor value and controls the motors. The idea is to keep the bot moving straight when it detects the line and to turn when it loses the line. Below is a basic example: 

if(sensor_detects_line) {
    // Move forward
} else {
    // Adjust direction until the line is found
}

Upload the code to your Arduino using the USB cable. Test the basic functionality by checking the sensor readings in the Serial Monitor.

Step 5: Testing and Calibration

After uploading the code, place your robot on a track with a clear, dark line on a light background. Observe the robot’s movement. You might need to adjust the sensor sensitivity or change the motor speeds in your code for better performance. Keep testing and fine-tuning until the bot follows the line smoothly.

Image: Testing the bot on a track

Arduino Code

The following code can be used to control the line following bot. Make sure the connections in the chart match your wiring. 

        
/* Simple Arduino Code for a Line Following Bot */

// Define pins for the IR sensors
#define leftSensorPin A0
#define rightSensorPin A1

// Define pins for the motor driver (for left and right motors)
#define leftMotorForward 8
#define leftMotorBackward 9
#define rightMotorForward 10
#define rightMotorBackward 11

void setup() {
  // Initialize the sensor pins as inputs
  pinMode(leftSensorPin, INPUT);
  pinMode(rightSensorPin, INPUT);
  
  // Initialize the motor control pins as outputs
  pinMode(leftMotorForward, OUTPUT);
  pinMode(leftMotorBackward, OUTPUT);
  pinMode(rightMotorForward, OUTPUT);
  pinMode(rightMotorBackward, OUTPUT);
  
  // Start serial communication for debugging (optional)
  Serial.begin(9600);
}

void loop() {
  // Read the values from the IR sensors
  int leftSensorValue = digitalRead(leftSensorPin);
  int rightSensorValue = digitalRead(rightSensorPin);
  
  // Print sensor values to the serial monitor (for debugging)
  Serial.print("Left Sensor: ");
  Serial.print(leftSensorValue);
  Serial.print(" | Right Sensor: ");
  Serial.println(rightSensorValue);
  
  // Decision making:
  // If both sensors detect the black line (LOW), move forward.
  if (leftSensorValue == LOW && rightSensorValue == LOW) {
    forward();
  }
  // If only the left sensor detects the line, turn left.
  else if (leftSensorValue == LOW && rightSensorValue == HIGH) {
    turnLeft();
  }
  // If only the right sensor detects the line, turn right.
  else if (leftSensorValue == HIGH && rightSensorValue == LOW) {
    turnRight();
  }
  // If no sensor detects the line, stop the bot.
  else {
    stopMotors();
  }
  
  delay(50); // Small delay for stable operation
}

// Function to move the bot forward
void forward() {
  digitalWrite(leftMotorForward, HIGH);
  digitalWrite(leftMotorBackward, LOW);
  digitalWrite(rightMotorForward, HIGH);
  digitalWrite(rightMotorBackward, LOW);
}

// Function to turn the bot left
void turnLeft() {
  // Stop left motor and run right motor forward to make a left turn
  digitalWrite(leftMotorForward, LOW);
  digitalWrite(leftMotorBackward, LOW);
  digitalWrite(rightMotorForward, HIGH);
  digitalWrite(rightMotorBackward, LOW);
}

// Function to turn the bot right
void turnRight() {
  // Run left motor forward and stop right motor to make a right turn
  digitalWrite(leftMotorForward, HIGH);
  digitalWrite(leftMotorBackward, LOW);
  digitalWrite(rightMotorForward, LOW);
  digitalWrite(rightMotorBackward, LOW);
}

// Function to stop both motors
void stopMotors() {
  digitalWrite(leftMotorForward, LOW);
  digitalWrite(leftMotorBackward, LOW);
  digitalWrite(rightMotorForward, LOW);
  digitalWrite(rightMotorBackward, LOW);
}

Conclusion

Building a line following bot with an Arduino Uno is a fun project that helps you learn about robotics and electronics. By following these simple steps, you can build your own bot and experiment with different improvements. I hope this guide was clear and helpful. Enjoy your project and keep learning!
