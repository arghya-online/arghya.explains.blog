LED Blinking System Using Arduino Uno

Objective

The goal of this project is to control an LED using an Arduino Uno by making it blink at a fixed interval. This is the most basic project that helps beginners understand microcontroller programming and circuit connections.

Items Needed

Item Quantity Description
Arduino Uno R3 1 The microcontroller board to control the LED
LED 1 Any color LED will work
Resistor (220Ω) 1 Limits current to protect the LED
Jumper Wires 2 Used to connect components

Figure 1: Circuit Diagram for LED Blinking Project

Arduino Code

// LED Blinking Code for Arduino Uno
int ledPin = 13;  

void setup() {
    pinMode(ledPin, OUTPUT);
}

void loop() {
    digitalWrite(ledPin, HIGH);
    delay(1000);
    digitalWrite(ledPin, LOW);
    delay(1000);
}
